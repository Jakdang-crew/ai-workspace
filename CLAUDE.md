This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 워크스페이스 개요

AI native 성장을 위한 학습/기록 공간. 각 프로젝트는 `projects/` 하위에 Git submodule로 관리됨.

## 작업 규칙

- 사용자와의 소통은 한국어로 진행
- 일지는 `journal/YYYY-MM-DD.md` 형식으로 작성
- 새 프로젝트 추가 시 submodule 사용: `git submodule add <url> projects/<name>`
